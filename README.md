# MultiEnrich

Notebook-first pipeline for multi-omics enrichment analysis on directionally coherent signals
(expression +  methylation), with GSEA leading-edge consensus.

## What this repo contains
- A Jupyter notebook (`notebooks/`) with the full workflow
- A bash runner to execute GSEA Preranked on multiple `.rnk` files (`scripts/`)
- Placeholder folders for inputs (`data/`) and outputs (`results/`)
